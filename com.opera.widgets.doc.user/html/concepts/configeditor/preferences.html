Preferences

The Preferences tab lets you define name-value pairs that will be associated with the widget the first time the widget is started.

To add a preference:

  1. Press the Add button. The dialog window will be opened.
  2. Type preference name and value.
    If the read-only flag is checked, the preference cannot be changed by a widget (in source code).
  3. Press the Finish button.

More information about preferences can be found at Preference - W3C Widgets Specification.
