Icons

The Icons shows icons associated with a widget.

To add an icon:

  1. Press the Add button. The dialog window will be opened.
  2. Press the Browse button and choose an appropriate icon.
    There is no need to set the icon's dimensions unless you want to change default values.
  3. Press the Finish button.

It is possible to drag an icon file from Project Explorer View and drop it into the icons list.
