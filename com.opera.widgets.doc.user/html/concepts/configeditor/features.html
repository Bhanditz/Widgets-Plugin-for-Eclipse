Features

The Features shows features what will be used by a widget.

To add a feature:

  1. Press the Add button. The dialog window will be opened.
  2. Type the feature URL.
    There is an autocomplete function in feature name, you need only to start typing a feature (e.g., "http://").
  3. Press the Finish button.

More information about features can be found at Features - W3C Widgets Specification.

If you want to use WAC features, please refer to: WAC Widget Handset API.
