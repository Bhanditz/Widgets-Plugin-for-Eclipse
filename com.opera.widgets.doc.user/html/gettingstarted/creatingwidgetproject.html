Creating the widget project

In this section, you will create a new widget project with the basic configuration.

Creating the project

  1. To create a new widget project go to File > New > W3C Widget Project. to open the New Widget Project wizard.
  2. Fill in the project name and press the Finish button.
  3. Go to Project Explorer to manage your newly created widget project.
  4. Click on src/config.xml to manage the widget configuration.
