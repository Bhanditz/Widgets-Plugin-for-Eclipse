Importing a widget package

If you want edit an existing widget (.wgt package), you can import it as a new project.

  1. To import a widget package into Eclipse, go to File > New > Import
  2. Choose Widgets/Widget package (.wgt) to open the Import Widget wizard.
  3. Click the Browse button and choose a widget file from the file system.
  4. Press the Next button.
  5. Choose a project name (by default it is based on the name from Widget config.xml file).
  6. Press the Finish button.
